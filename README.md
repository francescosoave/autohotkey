# autohotkey
volume controls + accents

Autohotkey commands:

--------------
Volume:

- F10: volume mute/unmute
- F11: decrease volume
- F12: increase volume


---------------
Italian accents (triple key press to trigger):

- à
- è
- ì
- ò
- ù


---------------
Italian accents (double key press):

- same characters as triple key press
- uncomment lines
